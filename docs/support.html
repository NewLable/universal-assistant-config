Frequently Asked Questions

Do I need to create an account?

No. Universal Assistant Pro does not require registration, login, a username, or a password. Open the app and start using it.

Does the app require an Internet connection?

Most features work offline. An Internet connection is required only for features that retrieve current online information, such as fuel prices, currency exchange rates, cryptocurrency and precious metal data, or weather-based atmospheric pressure.

Why does the app request location access?

Location access may be used by features such as Compass, Altimeter and Barometer when needed to provide their functionality.

Why does the app use device sensors?

Supported device capabilities are used only by relevant tools. For example, the magnetometer is used by Compass and motion sensors are used by Level and inclination tools.

Why does the app use the camera flash?

The device flash is used by Flashlight, Strobe Light, SOS Signal and Morse Code tools.

Does Universal Assistant Pro use AI?

No. Universal Assistant Pro does not use AI services.

Are there subscriptions, in-app purchases or advertisements?

No. The current version does not contain subscriptions, in-app purchases, premium upgrades or advertising.

Where can I read the Privacy Policy?

You can read the Universal Assistant Pro Privacy Policy.

Using the App

No setup is required. Launch Universal Assistant Pro, select a category from the home screen, choose a tool and use it.

Some features depend on hardware available on your device. If a particular sensor or capability is unavailable, the corresponding tool may have limited functionality.
